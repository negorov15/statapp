statapp: Metagenomic Statistical Analyzer

A user-friendly desktop application built with PyQt designed to streamline the processing and statistical interpretation of metagenomic datasets.

Key Features
Data Ingestion: Seamlessly load large-scale metagenomic files.

Analytical Framework: Perform Alpha/Beta diversity calculations, differential abundance testing, and correlation analysis.

Interactive Visualization: Real-time generation of PCoA plots, and taxonomic bar charts.

User-Friendly Interface: No command-line knowledge required; full GUI-driven workflow.
